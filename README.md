# Honeypot Attack Analysis Project

## Overview

This project presents an analysis of cyberattack activity observed using low-interaction honeypots deployed across multiple cloud platforms and geographic regions. The study investigates how infrastructure, location, and service exposure influence observed attack patterns.

The accompanying website provides access to the full report and supporting materials.

---

## 🔍 Project Focus

The research explores:

* Differences in attack activity across cloud providers
* Differences in attack activity across regions
* Commonly targeted ports and services
* Limitations of observed attack data in representing real attacker behaviour

## ⚠️ Notes

* The honeypots used were **low-interaction**, meaning they simulate services rather than fully operating systems

## 📌 Disclaimer
